# Py-100-Days-of-Learning

![python lint/format check](https://github.com/nlawren/Py-100-Days-of-Learning/actions/workflows/python-lint.yml/badge.svg?event=push)

A repository of python code and daily lessons/data/code following Dr Angela Yu's Udemy course of 100 Days of Code - Python

Reference link: [Course on Udemy](https://www.udemy.com/course/100-days-of-code/)

Python tooling is uv with python version 3.14 (as at March 2026). Using [prek](https://github.com/j178/prek) with ruff then github actions/ruff to format and lint at the repo level.
